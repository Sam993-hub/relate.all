# RelateAll Fullstack Setup (Firebase + React + React Native)

## 🌐 Web App Features
- Firebase Hosting
- Firestore DB for user profiles, messages
- Firebase Auth (Email/OTP/Google)
- Mutual Interest Matching (Swipe)
- Real-time Chat per Match
- Hindi/English language toggle
- Admin Dashboard (Moderate users/reports)

## 📱 Mobile App (React Native)
- Same Firebase backend
- Android APK ready
- iOS optional

## 🔐 Security
- Firebase Firestore rules
- Admin login with role check
- Profile visibility, data validation

## 📁 Folders
- `/web` – React frontend (MVP)
- `/mobile` – React Native app
- `/functions` – Firebase cloud functions (if needed)
